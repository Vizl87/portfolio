Sacred Run

A Dark Souls-inspired 2D Platformer

About the Game

This was the first game I created. The objective is to find bones scattered across the platforms. The game draws inspiration from the challenging nature of Dark Souls games and incorporates it into a classic platformer.

Technologies Used

GDScript Godot Engine Pixel Art

Key Features

  • Challenging platformer gameplay
  • Dark Souls-inspired difficulty
  • Custom pixel art graphics
  • Coin Collection
